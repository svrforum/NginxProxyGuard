## 🌐 DDNS 사용성 개선

v2.23.0의 프록시 호스트 ↔ DDNS 통합을 더 매끄럽게 다듬었습니다.

- **등록 즉시 자동 동기화** — 프록시 호스트에서 DDNS를 켜고 저장하면, 갱신 주기를 기다리지 않고 **바로 1회 동기화**합니다. 도메인이 곧장 현재 공인 IP를 가리킵니다. (동기화는 백그라운드로 처리되어 저장 속도에 영향을 주지 않습니다.)
- **설정 테스트에 DDNS 점검 추가** — 프록시 호스트 "설정 테스트"를 실행하면, DDNS가 켜진 호스트에 대해 **DNS 프로바이더 자격증명 검증**과 **마지막 동기화 상태(IP·시각·성공/실패)** 를 함께 보여줍니다. (원격 DNS는 변경하지 않는 읽기 전용 점검입니다.)
- **DDNS 메뉴 위치 이동** — DDNS 화면이 *설정* 에서 **인증서 → DDNS** 탭으로 옮겨졌습니다. DNS 프로바이더와 같은 메뉴에 모여 찾기 쉬워집니다.
- **프록시 호스트 폼 안내** — DDNS 자동 등록을 켜면 "인증서 → DDNS에서 상태를 확인하세요" 안내가 표시됩니다.
