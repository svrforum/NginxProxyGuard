## ✨ 프록시 호스트 UI/UX 개선

프록시 호스트 화면을 사용자 관점에서 다듬었습니다.

- **로드 밸런싱 발견성** — 프록시 호스트를 **새로 만들 때도 "로드 밸런싱(Upstream)" 탭이 보이며**, 호스트를 먼저 저장한 뒤 백엔드 서버·분배 방식·헬스 체크를 설정하라는 안내를 표시합니다. (기존에는 수정 화면에서만 노출돼 발견이 어려웠습니다.)
- **보안 탭 정리** — 보안 탭의 여러 섹션(우선 허용 IP, 취약점 차단, WAF, 접근 목록, 봇 필터, GeoIP, 클라우드 차단)을 **접을 수 있는 카드**로 바꿔, 사용하지 않는 섹션을 접어 원하는 설정을 더 빠르게 찾을 수 있습니다.
- **상태 표시 명확화** — 호스트 목록의 백엔드 상태 점 툴팁을 "백엔드 온라인/오프라인"으로 명확히 해, 활성/비활성(사용 여부) 토글과 구분되도록 했습니다.
- **실수 방지** — 호스트 행의 삭제 버튼을 다른 액션과 시각적으로 분리했습니다.
- **현지화 버그 수정** — 일부 한국어 화면에 영어로 노출되던 안내·툴팁(SSL 비활성 안내, 상태/기능 툴팁)을 번역했습니다.
