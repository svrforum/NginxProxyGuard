## 🩹 인증서 화면 카드 깨짐 수정 (#183, #185)

연결된 호스트가 많은 인증서에서 **인증서 카드가 세로로 길게 늘어나며 화면이 깨지던** 문제를 고쳤습니다.

### 🐛 수정
- **인증서 카드의 "연결된 호스트" 목록 캡 (#183, #185)**: v2.27.0 카드 리디자인 이후, 하나의 인증서에 연결된 프록시 호스트가 많을 경우(예: 와일드카드/다중 도메인 인증서) 호스트가 **제한 없이 세로로 쌓여** 카드가 비정상적으로 길어졌습니다. 이제 처음 **3개만 가로로** 보여주고, 나머지는 **"+N 더보기"**로 펼치고 **"접기"**로 다시 접을 수 있습니다 — 같은 카드의 도메인 목록과 동일한 방식입니다.

### 📝 참고
- 표시 방식만 바뀌었고 인증서·연결 정보 자체는 변동이 없습니다. 카드를 새로고침하면 바로 정상적으로 보입니다(브라우저 캐시 문제가 아니었습니다).
