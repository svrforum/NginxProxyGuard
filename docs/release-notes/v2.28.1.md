## 🛡️ 백신 오탐 유발 파일 제거 (OWASP CRS 테스트 픽스처)

Windows Defender 등 백신이 NPG의 `nginx_data/owasp-crs/tests/...` 파일을 **백도어(`Backdoor:PHP/Chopper`)로 오탐**하던 문제를 해결했습니다(#182).

### 🐛 수정
- **OWASP CRS 비런타임 파일 제거 (#182)**: CRS에는 WAF가 PHP 공격·웹셸을 제대로 차단하는지 검증하는 **회귀 테스트 픽스처**가 포함되는데, 여기엔 샘플 공격 페이로드(웹셸 문자열 등)가 들어 있어 백신이 이를 악성코드로 오탐했습니다. 이 파일들은 **런타임에 전혀 사용되지 않으므로**(실제 차단은 `rules/*.conf` 룰이 담당), 이미지와 볼륨에서 `tests/`·`util/`·`docs/`를 제거하도록 했습니다.
  - 백신 오탐이 사라지고 이미지도 가벼워집니다. **WAF 차단 동작에는 영향이 없습니다.**
  - 기존 설치도 **프록시 컨테이너 재시작 시** 볼륨에서 해당 파일이 자동 정리됩니다.

### 📝 안내
- 이전에 백신이 격리/삭제한 해당 테스트 파일은 실제 위협이 아니었으며(오탐), 제거돼도 NPG 동작에 영향이 없습니다.
