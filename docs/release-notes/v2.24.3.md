## ⚡ 로그 통계(차트·카운트) 캐싱 개선

대용량 환경에서 로그 화면의 **통계/차트 영역이 매번 다시 계산되던** 문제를 고쳤습니다.

- 날짜를 지정하지 않은 기본 화면에서 통계 캐시 키가 매 요청마다 달라져 **Redis 캐시가 전혀 적중하지 않던** 문제를 수정했습니다. 이제 같은 시간대의 반복 조회·자동 새로고침은 캐시된 결과를 즉시 사용합니다.
- 데이터가 많이 쌓인 환경(수억 건)에서 통계 영역의 반복 로딩이 **수 초 → 즉시** 수준으로 빨라집니다(최초 1회만 계산).
