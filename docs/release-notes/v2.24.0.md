## ✨ 전반적인 UI/UX 다듬기

인증서·WAF·로그·설정 등 주요 화면을 사용자 관점에서 전체적으로 손봤습니다.

- **부드러운 모달 애니메이션** — 모든 팝업(모달) 창이 갑자기 튀어나오지 않고 자연스럽게 페이드 + 확대되며 열리고 닫힙니다. 공통 모달 컴포넌트로 통일해 동작·접근성(ESC 닫기, 화면 리더 지원)을 일관되게 맞췄습니다. 입력 중인 폼은 바깥을 실수로 눌러도 닫히지 않아 작성하던 내용이 사라지지 않습니다.
- **설정 메뉴 정리** — 설정 화면의 탭이 10개가 제각각 색으로 흩어져 있던 것을, **일반 · 보안 · 운영** 세 그룹으로 묶고 구분선을 넣어 정리했습니다. 지나치게 포괄적이던 "보안" 탭은 실제 기능에 맞게 **"WAF 자동 차단"**으로 이름을 바꿨습니다.
- **한국어 현지화 보강** — 일부 화면에 영어로 노출되던 안내·버튼·옵션·툴팁(WAF 테스터 결과, 정책 변경 이력, 전역 설정 제목, DNS 프로바이더 오류, 로그 파일 화면 등)을 한국어로 번역했습니다.
- **실수 방지** — 전역 설정 초기화, WAF 규칙 일괄 비활성화처럼 되돌리기 어려운 작업에 확인 절차를 추가했습니다.
- **저장 결과 안내 통일** — 백업·시스템 로그·봇 필터 등 설정 저장 시 결과를 화면 상단 배너로 일관되게 보여줍니다(이전에는 일부가 팝업 경고창이었습니다).
- **일관성 개선** — 버튼·아이콘 색상을 기본 테마 색으로 통일하고, 아이콘 버튼에 접근성 라벨을 추가했습니다.
