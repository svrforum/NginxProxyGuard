## 🌐 DDNS — Dynu 프로바이더 추가

DDNS 자동 등록에 **Dynu**를 지원합니다. 기존 Cloudflare·DuckDNS에 이어 세 번째 프로바이더입니다.

- 인증서 메뉴에서 **Dynu** DNS 프로바이더(API-Key)를 등록한 뒤, 프록시 호스트의 **DDNS 자동 등록**에서 선택하면 해당 호스트의 도메인이 공인 IP 변경에 맞춰 Dynu를 통해 자동 갱신됩니다.
- 일괄 추가(프록시 호스트에서 추가) 모달에서도 Dynu 프로바이더를 선택할 수 있습니다.
- Dynu REST API(v2)를 사용하며, 동기화 상태·자격증명은 프록시 호스트 "설정 테스트"의 DDNS 섹션에서 확인할 수 있습니다.

**Full Changelog**: https://github.com/svrforum/NginxProxyGuard/compare/v2.23.1...v2.23.2
