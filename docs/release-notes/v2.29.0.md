## 🆕 UI에서 업데이트 확인 (#190)

관리 화면에서 **새 버전이 나왔는지 한눈에 확인**할 수 있습니다.

### ✨ 새 기능
- **업데이트 확인 카드 (설정 → 일반)**: 설치된 버전과 GitHub 최신 릴리즈를 비교해, **최신인지 / 새 버전이 있는지**를 표시합니다.
  - 새 버전이 있으면 **릴리즈 노트 링크**와 함께 업데이트 명령(`docker compose pull && docker compose up -d`)을 안내합니다.
  - "업데이트 확인" 버튼으로 수동 재확인이 가능합니다.

### 📝 참고
- **표시·안내 전용입니다 — NginxProxyGuard가 스스로 업데이트하지는 않습니다.** 컨테이너가 자기 이미지를 교체하려면 Docker 소켓 접근/다중 컨테이너 버전 조율이 필요해 보안상 위험하므로, 실제 업데이트는 위 명령으로 직접 적용합니다.
- 버전 확인은 GitHub Releases를 1시간 캐시로 조회합니다(레이트리밋 보호).
